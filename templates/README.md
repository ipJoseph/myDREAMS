# Templates Directory

This directory contains output templates for:
- Showing packages (HTML/PDF)
- Email templates
- Report templates

## Files (to be created)
- `showing_package.html` - Main showing package template
- `property_card.html` - Individual property card
- `email_template.html` - Email wrapper
- `styles.css` - Shared styles

## Assets
Place branding assets (logo, etc.) in `assets/` subdirectory.
